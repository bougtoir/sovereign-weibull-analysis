# Global Weibull Analysis of Sovereign Reign Durations

Data and analysis code for:

> **Global Weibull Analysis of Sovereign Reign Durations: A Reliability Engineering Approach to Quantifying Historical Political Stability**
>
> Submitted to *Humanities and Social Sciences Communications*

## Repository Structure

```
.
├── data/
│   ├── reign_durations.csv         # Reign durations for 31 polities (N = 1,148)
│   ├── violent_successions.csv     # Violent succession rates and civil war counts
│   ├── era_definitions.csv         # Historical era classification scheme
│   └── variable_definitions.csv    # Variable definitions and coding schemes
├── scripts/
│   ├── extract_data.py             # Data extraction and CSV export
│   ├── monarch_weibull_global.py   # Main Weibull analysis (31 polities)
│   ├── monarch_subanalysis.py      # Sub-analyses (elective vs hereditary; era-based)
│   ├── historical_validation.py    # Historical validation (violent successions, civil wars)
│   ├── accession_age_analysis.py   # Accession age distributions by polity
│   └── sensitivity_child_rulers.py # Sensitivity analysis excluding child rulers
└── figures/
    └── (generated by running analysis scripts)
```

## Data Description

### Reign Durations (`data/reign_durations.csv`)

Reign duration data for 31 sovereign polities spanning approximately 3,500 years of recorded history, compiled from established historical sources and verified against multiple references.

| Column | Description |
|--------|-------------|
| `polity` | Name of sovereign polity or dynasty |
| `reign_index` | Sequential index of reign within polity |
| `duration_years` | Reign duration in years |
| `region` | Geographic region (Europe, Asia, Middle East, Africa, Americas) |
| `succession_type` | Predominant succession mechanism (elective/hereditary) |
| `temporal_scope` | Time period analysed |
| `notes` | Additional context |

### Violent Successions (`data/violent_successions.csv`)

Independent dataset of violent succession rates (assassinations, depositions, coups) and civil war counts for 29 polities, used for historical validation of the Weibull shape parameter.

### Variable Definitions (`data/variable_definitions.csv`)

Complete variable definitions, coding schemes, and value ranges for all variables used in the analysis.

## Usage

### Export clean datasets

```bash
python scripts/extract_data.py
```

### Run main Weibull analysis

```bash
python scripts/monarch_weibull_global.py
```

### Run sub-analyses

```bash
python scripts/monarch_subanalysis.py
python scripts/historical_validation.py
python scripts/accession_age_analysis.py
python scripts/sensitivity_child_rulers.py
```

Generated figures are saved to the repository root directory.

## Data Sources

Reign duration data were compiled from the following sources:

- Wikidata SPARQL endpoint (structured queries for start/end dates of sovereign positions)
- Established historical references and encyclopaedias for verification
- Archigos dataset (Goemans et al., 2009) for modern political leaders
- Individual polity histories for specialised cases (see manuscript for full details)
